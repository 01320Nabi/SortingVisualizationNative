# SortingVisualizationNative

You can see how computer sort in different ways with this program.

Powered by OpenGL with GLFW

## How to use

1: 128 items  
2: 256 items  
3: 512 items  
4: 1024 items  
5: 2048 items  
tab: Change view mode  
b: Bubble sort  
s: Selection sort  
shift + s: Shell sort  
i: Insertion sort  
m: Merge sort  
q: Quick sort  
h: Heap sort  
scroll up: Increase interval  
scroll down: Decrease interval
